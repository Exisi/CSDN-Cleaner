# CSDN-Cleaner
## 前言
百度搜索的CSDN结果过于泛滥，此脚本主要针对CSDN的内容进行处理，尽可能的改善搜索体验

* 进入CSDN下载界面自动关闭
* CSDN文章界面下推荐中有关csdn下载的链接清除
* 百度搜索界面清除CSDN下载
* 百度界面搜索结果/相同文章去重
* 增加界面表格获取按钮，对csdn博客中的表格进行获取重绘，复制格式不混乱
* 防百度预加载干扰
